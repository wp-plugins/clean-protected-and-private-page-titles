Some themes prefix post titles with "Protected" or "Private"; this plugin removes those prefixes

== Description ==

Wordpress allows editors to mark Posts as "Protected" or "Private".

In some Themes, the words "Protected:" or "Private:" are automatically placed before the title of each post.

This plugin removes those words from the titles.

== Installation ==

1. Upload the "cleantitle" folder and its contents to the "/wp-content/plugins/" directory
1. Activate the plugin through the 'Plugins' menu in WordPress
